## Usage

Install:

```sh
# pnpm
pnpm add package-manager-detector

# npm
npm i package-manager-detector

# yarn
yarn add package-manager-detector
```

Import:

```js
// ESM
import { detect } from 'package-manager-detector'

// CommonJS
const { detect } = require('package-manager-detector')
```
