## Usage

Install:

```sh
# pnpm
pnpm add detect-package-manager

# npm
npm i detect-package-manager

# yarn
yarn add detect-package-manager
```

Import:

```js
// ESM
import { detect } from 'detect-package-manager'

// CommonJS
const { detect } = require('detect-package-manager')
```
